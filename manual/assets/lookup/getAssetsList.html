    # 设备查询

    功能介绍： 对系统设备进行管理，包括增加、删除、编辑等

    所在位置： 【设备管理 --> 设备查询】

    # 添加单台设备

    红色 * 的选项为必填项目，不填不能添加成功。蓝色 * 的选项为必填项目，可以后期补上，不填不影响添加

    # 批量添加设备

    批量添加设备需先下载模板文件，模板文件中红色标注的字段为必填字段

    # 批量修改设备

    不能通过批量修改设备功能修改设备所属科室，需要修改设备科室的，可以单台点击编辑，修改提交后，再经过审查批准同意后才会生效

    # 单台修改设备

    单台编辑修改设备，除了修改科室之外，修改其他属性值均立即生效

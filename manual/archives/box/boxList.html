    # 档案盒管理

    功能介绍： 创建、编辑档案盒

    所在位置： 【档案管理 --> 档案盒管理】

    页面介绍

    档案盒管理目的：把某一类或某一科室的设备档案归类到一个档案盒下面，方便对某类设备的档案进行管理

    # 添加档案盒

    1、点击“添加档案盒”，弹出一个新页面

    2、填写档案盒名称

    3、填写档案盒编号（自定义，不允许重复）

    4、先选择科室，科室选择后会在右边下拉列表中返回该科室设备，然后选择需要绑定到该档案盒的设备即可

    5、填写备注，点击“保存”按钮即可新建档案盒成功

    可以在此进行档案盒修改操作。修改操作类似添加操作，这里不做介绍

    设备查询列表中查询急诊科-空气消毒机，打开其中一个设备的详情，可以看到已经绑定了相关的档案盒

    在详情页面中切换到“设备档案”，点击“本地上传”按钮，上传一个关于该设备的文件

    在档案盒列表中可以看到，该档案盒的文件数量变为 1

    点击“查看”按钮，查看档案盒里面的内容，在这里可以看到我们刚刚上传的急诊科空气消毒机(1512702530)设备的文件了

    点击“查看”按钮，查看档案盒里面的内容，在这里可以看到我们刚刚上传的急诊科空气消毒机(1512702530)设备的文件了

    可以在添加、修改主设备的时候，选择相应的档案盒

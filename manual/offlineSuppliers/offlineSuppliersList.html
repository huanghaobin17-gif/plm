    # 厂商列表

    功能介绍： 新增、修改、删除厂商数据

    所在位置： 【厂商管理 --> 厂商列表】

    页面介绍

    厂商管理目的：把某一类或某一科室的设备档案归类到一个档案盒下面，方便对某类设备的档案进行管理

    # 添加厂商

    1、点击“添加厂商”按钮，打开一个新页面

    2、填写企业基本信息

    3、上传企业相关资质

    4、点击提交即可新建厂商

    批量上传设备时，如果设备中的厂商不存在于系统中，也会自动把该厂商添加进系统中

    厂商的类型有供应商、生产商、维保商、维修商四种，厂商类型可体现在添加主设备、维修设备的时候供不同的选择

    如刚添加的 不属于生产商，故在“生产厂商”下拉列表中没该公司选项，在供应商列表中则有此选项

    在【维修管理 -- 维修业务管理 -- 维修处理 -- 检修】中，选择第三方维修时，填写维修报价公司时，公司列表中，属于“维修商”类型的厂家才可以进行选择

    # 批量修改厂商类型

    1、选择需要批量修改的厂商

    2、点击“批量修改厂商类型”

    3、弹出一个弹窗，勾选需要更改的类型

    4、确认提交即可修改厂商类型
